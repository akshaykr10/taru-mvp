05 · Foundation
Motion & Easing
Motion is purposeful and emotionally calibrated. The parent app moves with calm confidence. The kid app celebrates with delight. Never animate for decoration.
Easing Tokens
TokenValueUse case
--ease-brandcubic-bezier(0.16, 1, 0.3, 1)Parent app screen transitions, modals
--ease-bouncecubic-bezier(0.36, 0.07, 0.19, 0.97)Kid app celebrations, badge pops
--ease-springcubic-bezier(0.34, 1.56, 0.64, 1)Button press, toggles, checkmarks
--ease-out-softcubic-bezier(0, 0, 0.2, 1)Drawers, tooltips, dropdowns
Duration Tokens
TokenDurationUse case
--dur-instant80msHover state color fills, icon changes
--dur-quick150msButton press, badge appear, checkmark
--dur-standard250msScreen transitions, modals, overlays
--dur-deliberate400msXP bar fill, chart draw-in, number count-up
--dur-celebration600–800msAchievement unlock, level-up sequence
Animation Principles
Parent App

Numbers count up smoothly. Screens slide or fade. Nothing bounces or jiggles. Portfolio values should feel trustworthy — precision > delight here.
Kid App

Achievements bounce in. XP bars fill with a gleam. Coins fly on reward. Gilli floats gently when idle. Every interaction should feel rewarding.

Mascot Rules
Personality
Optimistic but honest. Uses simple analogies. Never condescending. Celebrates small wins as much as big ones. Never uses adult financial jargon with kids.
Sizing
Full-body (80–100px): celebration. Head/bust (48–60px): tips and nudges. Icon (24–32px): brand marker. Never in transaction flows or dashboards.
App Context
Kid app: Gilli is always present — guides, celebrates, teaches. Parent app: Gilli appears only for insights, milestones, and educational nudges.
⚠ Restrictions
Never show Gilli sad, scared, or disappointed. No negative emotional states. Never during error flows or transaction failures.
